How it works

MacHive wraps the open-source exo engine in a simple menu bar app. You get the power of distributed inference without touching the terminal.

1

Install on each Mac

Download the DMG, drag to Applications, and launch. MacHive installs Homebrew, Python, uv, Node, and exo automatically. One-time setup takes 10–30 minutes.

2

Connect the cluster

Macs find each other on the same WiFi using Bonjour, UDP broadcast, and UDP multicast. No IP addresses to type. No router config.

3

Chat with big models

Click Start AI Cluster on every Mac. When status shows Ready, click Open Chat. Your Macs split the model layers and run inference together.

Perfect setup checklist

Do this on every Mac you want in the cluster.

  • macOS 13 or later on Apple Silicon (M1/M2/M3/M4)
  • MacHive.app is in /Applications
  • Allow Local Network access when macOS asks
  • Turn OFF macOS firewall: System Settings → Network → Firewall
  • All Macs on the same WiFi network
  • Click Start AI Cluster on every Mac
  • Wait for Ready, then click Open Chat

Common questions

Does it work with two Macs?▼

Yes. Two Macs is the minimum. Add more Macs and the combined RAM grows, letting you run bigger models.

Do I need to be an Administrator?▼

Yes, for first-time setup. MacHive installs Homebrew and other tools, which requires an admin password. After setup, normal use does not need admin.

Can I use Ethernet instead of WiFi?▼

All Macs must be on the same local network. Ethernet works if the Macs are on the same subnet. Avoid mixing Ethernet + WiFi unless they share the same network.

What if setup fails?▼

MacHive shows a manual install command. Copy it, paste it in Terminal, and enter your Mac password when asked. It does the same steps the app does.
