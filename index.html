Perfect setup checklist

Do this on every Mac you want in the cluster.

  • macOS 13 or later on Apple Silicon (M1/M2/M3/M4)
  • MacHive.app is in /Applications
  • Allow Local Network access when asked
  • macOS firewall is OFF
  • All Macs on the same WiFi network
  • Click Start AI Cluster on every Mac
  • Wait for Ready, then click Open Chat
